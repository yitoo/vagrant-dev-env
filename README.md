Yitoo dev en 
=================

Requirements
----
- vagrant + shell + virtualbox
- ubuntu/xenial64
- php7
- nginx

How to install ?
--------------
In the host machine, ensure vagrant and virtualbox are installed

```shell
$ git@github.com:yitoo/vagrant-dev-env.git yitoo-vagrant-dev-env
$ cd yitoo-vagrant-dev-env
$ vagrant up --provision
```
